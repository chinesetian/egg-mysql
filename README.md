# egg-mysql
## egg连接mysql

1. 后端渲染
- 实现mysql数据库的连接
- 查询数据库数据并用html模板渲染

2. 后端服务
- 实现登录接口
- 实现增删改查接口



### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 生产构建

```bash
$ npm start
$ npm stop
```

### 其他

1. mysql 5.7
2. Navicat Premium 12 数据库连接工具
